# 🚀 Projet TypeScript - Exercices (Episodes 1 à 13)

## 📌 Présentation du projet

Ce projet est une série de 13 exercices réalisés en **TypeScript** avec **Vite**.  
Il a pour objectif de pratiquer progressivement les bases de la programmation.

Chaque épisode aborde une notion différente : variables, conditions, boucles, calculs et interactions utilisateur.

---

## 🎯 Objectifs pédagogiques

Ce projet permet de :

- Comprendre la syntaxe TypeScript
- Manipuler les variables et les types
- Utiliser les conditions (`if / else`)
- Utiliser les boucles (`for`, `do...while`)
- Travailler avec les fonctions natives (`prompt`, `alert`, `console.log`)
- Développer une logique algorithmique progressive
